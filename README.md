#  Handwire-keyboard-configurator

## Prerequisites

  1. Install [Rust, Cargo](https://www.rust-lang.org/tools/install)

## Run Locally

Clone the project

```bash
  git clone git@github.com:Spheny1/handwire-config-builder.git
```

Build and run

```bash
  cd lambda
  cargo lambda run --release
```
Connect to localhost:3000
